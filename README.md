# ZhouYM Personal Homepage

这是一个可直接部署到 GitHub Pages 的静态个人主页，用于整理数学研究兴趣、论文阅读和学习笔记。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 发布到 GitHub Pages

1. 在 GitHub 新建仓库，推荐命名为 `zhouym222.github.io`。
2. 把本目录中的 `index.html`、`styles.css` 和相关笔记文件上传到仓库根目录。
3. 打开仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/root`，保存后等待部署完成。

部署成功后，主页通常会出现在：

`https://zhouym222.github.io/`

如果 GitHub 用户名不是 `zhouym222`，请同时修改 `index.html` 里的 GitHub 链接和上面的地址。
